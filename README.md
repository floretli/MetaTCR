


## Installation

dependencies:

```
tqdm
scipy
biopython
matplotlib
touch >= 1.1.0 (tested on 1.7.1) 
pandas 
numpy 
sklearn

faiss-gpu
```

```
02_calculate_repertoire_vectors.py --dataset_dirs ./data/example_repertoire.datalist --centroids_file ./results/data_analysis/96_best_centers.pk
```
